!! SAFETY TIP: Always inspect a BAT file in Notepad before running it !!

A BAT file that uses tes3cmd to delete exterior and interior changes to Seyda Neen from Beautiful Cities of Morrowind, so that you can run both BCOM and Seyda Neen Damp Little Squat.

Make sure you have tes3cmd and Beautiful Cities of Morrowind installed in Data Files

Then run this BAT file by double clicking it.



MO2 users
-----------------
You need to add this BAT file to your executables list.

1. Go to the executables menu
2. Hit the + icon to add an executable.

Title: BCOM SNDLS compatibility BAT
Binary: C:\Steam\steamapps\common\Morrowind\Data Files\BCOM_SNDLS_compat.bat
Start in: C:\Steam\steamapps\common\Morrowind\Data Files

Change C:\Steam\steamapps\common\Morrowind\Data Files to wherever your Morrowind\Data Files folder is stored.
